# Water Heater Corrosion Protection

Technical documentation and information bulletins for electronically controlled cathodic corrosion protection systems used in water heaters and industrial storage tanks.

## Technologies
- Active titanium anodes
- Impressed current cathodic protection (ICCP)
- MMO-coated electrodes
- Potentiostat control systems
- Corrosion prevention for enamelled and stainless steel tanks
